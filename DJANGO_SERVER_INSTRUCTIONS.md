# تعليمات تشغيل Django Server

## للسماح بالاتصال من Flutter (جهاز فعلي أو Emulator)

### الطريقة 1: من Terminal

```bash
cd smartju
python manage.py runserver 0.0.0.0:8000
```

**ملاحظة:** استخدم `0.0.0.0:8000` بدلاً من `127.0.0.1:8000` للسماح بالاتصال من الشبكة.

### الطريقة 2: استخدام ملف Batch (Windows)

شغّل `START_DJANGO.bat` من المجلد الرئيسي.

### الطريقة 3: استخدام PowerShell Script

شغّل `START_DJANGO.ps1` من PowerShell.

## الفرق بين `127.0.0.1` و `0.0.0.0`:

- **`127.0.0.1:8000`**: يستمع فقط على localhost (لا يمكن الوصول من الشبكة)
- **`0.0.0.0:8000`**: يستمع على جميع واجهات الشبكة (يمكن الوصول من الشبكة)

## بعد التشغيل:

سترى:
```
Starting development server at http://0.0.0.0:8000/
```

الآن يمكن الوصول من:
- `http://127.0.0.1:8000` (من نفس الجهاز)
- `http://192.168.43.198:8000` (من الشبكة المحلية)
- `http://10.0.2.2:8000` (من Android Emulator)

## إيقاف السيرفر:

اضغط `CTRL+C` في terminal.
